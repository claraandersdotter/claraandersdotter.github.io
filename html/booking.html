A booking system

Type: Individual work

When: 2022

Duration: Two weeks, half time

Project scope: School project

The objective of this project was to develop an interface for a booking system that included both rooms and equipment. A Figma mockup was created to visually represent the layout and features of the booking system. Both students and teachers provided feedback on design decisions to guarantee alignment with their needs.
